# Mety Chatbot Frontend

Frontend for the Mety Chatbot Assistant Platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Configuration

Set environment variables:
- `VITE_BACKEND_URL` - Backend API URL (default: http://localhost:8000)
- `VITE_PREDICTION_API_URL` - Prediction API URL (default: https://mlmodel.myyouthspan.com/prediction_model_test/)

## Features

- Onboarding flow (4 pages)
- Plan management with prediction API integration
- Log submission with adherence calculation
- Chat interface with variable extraction
- Direct prediction API connection from frontend
